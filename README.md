# **上班摸鱼写作软件**

*最快使用方法：直接下载writing.exe到本地，就可以直接使用了。*

功能特性：（1）每隔1.5秒保存一次，即使你电脑突然关机，下次打开也会保留最后一次编辑内容
（2）快捷键导出，可自定义导出格式
（3）快捷键调整透明度，且鼠标离开界面五秒，自动变灰暗。
